# Life Adventure Simulator

A browser-based life simulation game where you can experience various aspects of life including career, relationships, business, crime, and more.

## Features

- **Career System**: Work different jobs with varying salaries
- **Education**: Study to increase intelligence for better opportunities
- **Business & Investments**: Start businesses for passive income
- **Real Estate**: Buy properties for rental income
- **Vehicle Collection**: Purchase cars with different stats
- **Relationships & Family**: Date, marry, and have children
- **Banking System**: Deposit, withdraw, and take loans
- **Stock Market**: Invest in volatile stocks
- **Crime System**: Engage in illegal activities with risks
- **Inventory System**: Collect and manage items
- **Random Events**: Experience unexpected life events

## How to Play

1. **Start Simple**: Begin with basic jobs to earn money
2. **Invest Wisely**: Put money into businesses and properties
3. **Maintain Stats**: Keep health, happiness, and energy levels up
4. **Take Risks**: Gamble or commit crimes for quick money (but be careful!)
5. **Build a Life**: Get married, have children, buy houses and cars

## Keyboard Shortcuts

- `Ctrl + S`: Save game
- `Ctrl + L`: Load game
- `F1`: Show help

## Mobile Support

The game is fully responsive and works on mobile devices with a touch-optimized interface.

## Deployment on Render

1. Create a new Web Service on Render
2. Connect your GitHub repository
3. Use the following settings:
   - **Build Command**: (leave empty - static site)
   - **Start Command**: (leave empty)
4. The site will be deployed automatically
